Дожим за 3 дня: не подписал на первом звонке

Потолок «контакт → договор» — 3 дня активного времени, абсолютный (свод §0). Дальше лид уходит другому с мерджем. Значит, каждый день дожима обязан двигать сделку, а не «напоминать о себе». Ниже — план по дням: каждое касание с целью, готовым текстом и инструментом.

Попытка = телефонный набор. Сообщение — дополнение, окно 48 ч само по себе не закрывает.
Каждое касание — новая ценность. Рубрика звонка: попытался закрыть · отработал возражение · новый шаг с датой. 0 из 3 = «пустой дожим», работой не считается (§10).
Ритм: максимум 1 звонок + 2 сообщения в день. Больше — уже давление, растёт риск отказа.
«Прекратите контакт» = стоп. Отказ с цитатой → закрытие с причиной, DNC. Право закрыть есть.
КИ пришла — план дня отменяется. В любой день: КИ в виджет → заключение → немедленно звонок «разбор». Это самый сильный момент.

Конец дня 3 — потолок. Что происходит автоматически

  • Лид уходит другому менеджеру с мерджем всей истории (свод v2.11) — новые касания начнёт коллега, уже с заключением и перепиской.
  • Финальный вердикт AI-QA догоняет: живой клиент + недобитый гейт = −1 жизнь; недозвоны рейтинг не трогают.
  • КИ так и не запрошена за 3 дня = провал обязательного шага (канон «не купил → запрос КИ обязателен»).
  • Клиент вернулся сам после потолка → тёплый пул, приоритетная выдача.

Инструменты под рукой

  • Кнопки ✉ в amo-виджете — «Не открыл» / «Не подписал» / «По КИ»: тексты с цифрами клиента подставляются сами. Остальные шаблоны — копируйте с этой страницы (плейсхолдеры {…} заменит виджет будущего релиза; пока — руками).
  • Инструкция клиенту по КИ — «Кредитная история за 5 минут» (scoring.ru через Госуслуги, бесплатно, «ничего не покупать»).
  • Пример юр. заключения — обезличенный образец: показывайте до получения КИ, после — шлите персональное.
  • Опция на будущее: голосовое/кружок от юриста «посмотрел вашу историю» перед звонком №2 — персонализация сильнее любого текста (в прод-релизе).
  • Помнить: подписал, но не оплатил — это уже другой режим (ст. 5: попытка каждый день, потолка нет, частичка-добор). Этот плейбук — только про «не подписал».
