Полная карта: канонические окна §0, схема дожима Виктории (с решениями 22.07: скидка-ускоритель, реферальная, ki_refused, частичка от 990) и ⊕ дополнения — дыры из сверки сценариев, закрытые предложением. менеджер система / движок деньги ⊕ дополнение — на утверждение закрытие пунктирная стрелка = ⊕ / возвратный путь

⊕ Что дополнено против схемы Виктории и текущих доков

Каждый пункт закрывает дыру из сверки сценариев (карта состояний, 22.07). Это предложения — вносим в канон по утверждению.

  1. Мост дожима до фазы 2: пока авторассылки нет, касания Д0 выдаёт Sensei-задачей с готовым нередактируемым текстом — иначе дожим первички не шлёт никто (P0).
  2. Недозвон на назначенный звонок дожима (Д1, «да-нет», разбор по КИ): попытка + замещающее сообщение + сдвиг дня цепочки. Сейчас не описан нигде.
  3. Ведение ст.5 после первых суток: касания через день с контентом (гарантия п.6 → бронь юриста → «что мешает» + перенос на 25-е), а не пустые пинги; лимит переносов «оплачу завтра» = 2, дальше разбор РОПа.
  4. Откат скидки-ускорителя: не оплатил до 23:59 → событие discount_expired, страница пересчитывается обратно, одно касание «дедлайн прошёл — условия обычные». Без этого «только сегодня» = фикция.
  5. Передумал после подписи (до денег): разбор «почему» по карточке возражения №14 → вернули к оплате или расторжение с цитатой. Ветки в доках не было.
  6. КИ хуже слов: дельта >15% → холд авторассылки + задача «звонок-разбор» до любого автосообщения с новым планом.
  7. Парсер КИ не справился: путь «по словам» + файл юристу, НЕ закрытие клиента (false-negative ≠ отказник).
  8. После оплаты №1: вечная ссылка становится milestone-страницей, welcome-звонок ≤24 ч, pre-dunning за 3 дня до 2-го платежа, пропуск → каскад Д1-Д3-Д7 + эскалация клоузеру в день сигнала (его вторые 50% висят именно тут).
  9. Расторжение/возврат: события contract_terminated / refund в движке; страница перестаёт показывать «Готово»; разбор — по возражению №14, твёрдое «расторгаю» — принимаем без дожима.
  10. Горячий сигнал (открыл страницу / КИ / чат): в рабочие часы — звонок владельца ≤30 мин; владелец офлайн → горячая очередь. Раньше правило было только для хвоста.
  11. Возврат закрытых (недозвон / «не наш») через 30–90 дней: владелец рецикла — движок: пакетом в общий пул с тегом «возврат», холодная каденция, НЕ прежнему менеджеру.
